## Simple Description

We know that it is difficult to express a beautiful aurora on the web.

This interferes with sync between UIUX designers and web front-end developers.
We devised the easiest way to express the beautiful aurora, and completed the product in two days for a quick solution.

I recommend automatic for the best beautiful color for the color you choose. The recommended color is the result of mathematically calculating saturation and brightness, and may look beautiful.

You can also choose the size of the components to apply this beautiful aurora. We export the code according to your settings.

Finally, our products support various forms for more people to use in a variety of ways. This can be found in [Supported formats](#supported-formats).
